## 安装/使用:

遵循以下4个简单的步骤, 就能使用demo

**步骤 1**: Clone 内容
```bash
$ git clone https://github.com/guless/bingoJS2.git bingoJS2
```

**步骤 2**: 进入bingoJS2目录
```bash
$ cd bingoJS2
```

**步骤 3**: Install dependencies
```bash
$ npm install
```

**步骤 4**: 启动demo
```bash
$ npm start
```
### 其它:
**src目录**：代码开发目录, 如果代码变化自动生成到dist目录

**dist目录**：最终生成目录
